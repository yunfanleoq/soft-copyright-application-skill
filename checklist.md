# 软件著作权登记 — 申报材料勾选清单

> 办理机构：中国版权保护中心（[ccopyright.com.cn](https://www.ccopyright.com.cn/)）  
> 勾选 □ → 完成 ■

---

## §1 必备材料（所有情形）

| □ | 材料 | 说明 |
|---|------|------|
| □ | 软件著作权登记申请表 | 在线填写后打印/导出；申请人签章 |
| □ | 源程序鉴别材料 | **PDF**；前 30 + 后 30 页（不足 60 页交全部）；≥50 行/页；末页≥15 行 |
| □ | 文档鉴别材料 | **PDF**；用户操作手册（推荐）；前 30 + 后 30 页；≥30 行/页；**实机截图** |
| □ | 申请人身份证明 | 见 §2 |

---

## §2 身份证明（按申请人身份勾选一项）

| 申请人身份 | □ | 材料 |
|-----------|---|------|
| 企业法人 | □ | 营业执照副本复印件，**加盖公章** |
| 事业法人 | □ | 事业单位法人证书副本复印件，**加盖公章** |
| 自然人 | □ | 身份证正反面复印件，**签字** |

---

## §3 权属与特殊情形证明（按实际情况勾选）

| 情形 | □ | 额外材料 |
|------|---|----------|
| 独立开发（默认） | □ | 无额外权属合同 |
| 合作开发 | □ | 合作开发合同或协议；全体著作权人身份证明 |
| 委托开发 | □ | 委托开发合同；委托方/受托方身份证明 |
| 下达任务开发 | □ | 项目任务书或合同 |
| 修改他人已登记软件 | □ | 原著作权人**书面许可证明** |
| 继受（继承/受让/承受） | □ | 继受证明文书 + 原登记证书复印件 |
| 法人变更/终止后承受权利 | □ | 工商变更证明、承受协议等 |

---

## §4 例外交存（可选，涉密时使用）

> 仅在商业秘密确实需要遮盖时使用；须符合登记办法第十二条比例限制。

| 方式 | □ | 要点 |
|------|---|------|
| 方式一 | □ | 源程序前、后各 30 页；机密部分黑斜线覆盖，覆盖 ≤50% |
| 方式二 | □ | 前 10 页 + 任意连续 50 页 |
| 方式三 | □ | 目标程序前、后各 30 页 + 源程序任意连续 20 页 |
| 文档例外交存 | □ | 参照源程序例外交存规则 |

---

## §5 PDF 格式与打印（详见 pdf-format-spec.md）

| □ | 检查项 |
|---|--------|
| □ | 源程序、文档鉴别材料均为 **PDF**（非仅 Word） |
| □ | A4、纵向、**单面**；鉴别材料 **黑白** |
| □ | 页眉左：软件全称 + 版本号（与申请表一致） |
| □ | 页眉右：**右上角**连续页码（源程序 1～60 或全文；文档自目录/正文起） |
| □ | 页脚中：**著作权人全称** |
| □ | 源程序：等宽字体，每页 ≥50 行有效代码 |
| □ | 操作手册：登录页截图 + 分步操作截图；每步有文字说明 |
| □ | 截图内软件名、版本、著作权人与申请表一致 |
| □ | PDF **无加密**、无第三方水印；单文件 <50MB |
| □ | 申请表在线生成，**不擅自改表格式** |
| □ | 外文证明附 **中文译本** |

---

## §6 一致性（提交前必查）

| □ | 检查项 |
|---|--------|
| □ | 软件全称在各材料中完全一致 |
| □ | 版本号在各材料中完全一致（含是否带 `V`） |
| □ | 著作权人名称与营业执照/身份证一致 |
| □ | 开发完成日期 ≤ 申请日期 |
| □ | 源程序体现核心功能，非仅配置/依赖清单 |
| □ | 已剔除 `.env`、密钥、真实客户数据 |
| □ | 使用开源组件已说明并备许可协议（如适用） |

---

## §7 线上提交文件命名建议

```text
{申请人全称}-{软件全称}{版本号}-申请表.pdf
{申请人全称}-{软件全称}{版本号}-源程序.pdf
{申请人全称}-{软件全称}{版本号}-操作说明书.pdf
{申请人全称}-营业执照.pdf
```
