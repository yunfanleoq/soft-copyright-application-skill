# 操作手册 PDF 排版规范

生成 `操作说明书.md` → PDF 时须遵守本规范，避免提前换行、表格错位、截图过小等问题。

## 1. Markdown 写法

### 1.1 小节导语（`**标签** 正文`）

使用 `**标签** 正文` 分段，**不要**把多段合并为一行：

```markdown
**工作模式** 左侧导航、案件信息栏同时显示……

**使用便利** 一屏掌握案件全貌……
```

解析器会将每个 `**标签**` 拆为独立段落，标签后自动补全角冒号 `：`。

### 1.2 仅标题行（表格/列表前）

```markdown
**Agent 执行顺序一览：**

| 步骤 | 界面阶段 | … |
```

单独一行的 `**xxx**` 或 `**xxx：**` 会渲染为小标题，后接表格。

**表格前标签须带一句说明**，不要只写空标签：

```markdown
**字段权限** 系统项编辑时各字段的可改范围如下表所示。

| 字段 | … |
```

否则 PDF 中可能只出现「字段权限：」而无表格（旧版引擎未在截图页渲染表格）。


### 1.3 禁止误用

| 错误 | 后果 |
|------|------|
| 多段 intro 写成一行用空格连接 | PDF 在 `OA`、`Agent` 等英文后提前换行 |
| 表格续行第一列只写 `①` | 只剩序号、看不出阶段名 |
| 子节标题 `5.4（1）① xxx` | 目录/标题显示不完整 |
| `**标签**` 单独一行后直接接表格、无说明 | PDF 仅显示空标签（须补一句导语） |
| 截图带红框 overlay | 不符合实机截图要求 |

### 1.4 步骤与预期结果

仍用 `**步骤 1** …`、`**预期结果** …`，由专用规则解析。

### 1.5 图下说明（截图页）

每张截图后、步骤前，可写 `**图下说明**` 段落，PDF 中渲染顺序为：**节首导语 → 截图 → 图注 → 图下说明 → 步骤**。同一节多张截图时，每张图各自配一段图下说明。

```markdown
![5.22 划词快捷提问](screenshots/.../02-selection-menu.png)

**图下说明** 如上图所示，选中文字后弹出浮动工具条……
```

## 2. PDF 引擎（`generate_manual_pdf.py`）

### 2.1 段落合并

- intro 块以 `\n\n` 分隔，`write_body` 逐段排版
- `**标签**` 行不再与上一段用空格拼接

### 2.2 防提前换行（`_protect_phrase_breaks`）

以下词组内空格替换为不换行空格（NBSP）：

- `IP Agent`、`OA`、`WOA`、`Agent`、`D1/D2`、`OA/PDF`
- `读 → 比 → 写 → 审` 箭头链；`导航 / 案件信息 / AI` 斜杠链
- `隐藏 + 工作流` 加号链
- 拉丁字母/数字 + 中文、中文 + 拉丁（双向）
- 书名号 `「…」` 内不断行；`图 5.2-1` 图号；`Enter 键`；`8 GB`

新增易断词组时，优先加入 `_NO_BREAK_PHRASES` 或 `_protect_phrase_breaks` 规则。

### 2.3 中文禁则（`wrap_text_kinsoku`）

fpdf 原生 `multi_cell` 无 CJK 禁则。引擎已改为 **`wrap_text_kinsoku` 手动换行**，规则包括：

| 规则 | 说明 |
|------|------|
| 行首禁则 | `、` `。` `，` `；` `：` `！` `？` 及闭括号等不得单独出现在行首 |
| 行末禁则 | `（` `「` `《` 等开括号不得单独留在行末 |
| 孤标点合并 | 若某行仅剩句号/顿号等，**自动并入上一行**（用户要求：句号不能独占一行） |
| 英文词组 | `Reviewer Agent`、`IP Agent` 等空格改 NBSP，不得中间断行 |

标点前另插入 NBSP 作为双保险。表格单元格、正文、步骤、标题、图注均走同一套换行逻辑。

### 2.4 截图页表格与留白

- `write_image_page` 会在导语后渲染 `table_rows` / `extra_tables`（如 5.19（4）字段权限表）
- 判断换页时**只预留截图+图注高度**，步骤/预期结果可续页，避免 5.22 等节导语与截图之间出现大块空白

### 2.5 同行多段 intro

解析器会将同一行多个 `**标签** 正文` 自动拆为独立段落；**仍建议在 MD 中每段单独一行**，便于 diff 与校验。

### 2.6 表格列宽

| 列数 | 用途 |
|------|------|
| 4 | WOA Agent 顺序表 |
| 8 | 字典权限对照表 |
| 3 | 折叠开关说明表 |

### 2.7 截图页

- intro 过长导致图片高度不足时，**自动换页**后再插图
- 图片最小高度约 95mm，尽量占满 `epw` 宽度
- 步骤/预期结果可跨页（图后 `remaining_y < 14` 时续页）

## 3. 发布前检查

```powershell
# 截图文件齐全
py -3.12 %USERPROFILE%\.cursor\skills\soft-copyright-application\scripts\validate_manual_screenshots.py --project-root "<项目根>"

# 换行风险 + MD 结构（同行多 intro、表格首列仅序号等）
py -3.12 %USERPROFILE%\.cursor\skills\soft-copyright-application\scripts\validate_manual_linebreaks.py --project-root "<项目根>"

# 生成 PDF（项目内脚本已串联上述两项）
powershell -ExecutionPolicy Bypass -File soft-copyright\scripts\build_manual_pdf.ps1
```
